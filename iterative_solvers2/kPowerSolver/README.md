# Power method with deflation

## Arguments

* arg1: Ruta archivo matriz A
* arg2: Number of eigenvetors and eigenvalues to obtain

## Example

- >> make
- >> ./runTest ../M_BIG.txt 5
- >> make clean

## Output

- k valores propios más grandes
- k vectores propios correspondientes

* [C11] en Ubuntu 16.04 LTS
