# Bisection method

## Arguments

* arg1: Número de función: 1, 2, 3, 4, 5, 6
* arg2: Inicio intervalo: Double
* arg3: Fin intervalo: Double

## Example

- >> make
- >> ./runTest 1 -2 2
- >> make clean

## Output

- Raíz encontrada
- Gráfica de función y raíz encontrada
